## 💻 Local Development
Since this project uses pure HTML and CSS, no package manager or build steps are required. 
1. Clone the repository: `git clone https://github.com/yourusername/portfolio-website.git`
2. Navigate to the project directory: `cd portfolio-website`
3. Open `index.html` directly in any modern web browser.

## ☁️ Deployment
This portfolio is configured for immediate deployment on **Vercel**.
1. Push your code to a GitHub repository.
2. Import the repository into your Vercel dashboard.
3. Vercel will automatically detect the static files and deploy them. Any future pushes to the `main` branch will trigger an automatic rebuild and deployment.
